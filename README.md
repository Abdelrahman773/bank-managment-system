# Bank Management System

A full-stack banking management system that enables users to securely manage their accounts through a modern web interface. The application provides essential banking operations such as deposits, withdrawals, and money transfers while implementing secure authentication using JSON Web Tokens (JWT).

## Features

- Secure user registration and login
- JWT-based authentication and authorization
- Deposit funds
- Withdraw funds
- Transfer money between accounts
- Account balance management
- Transaction validation
- RESTful API architecture
- Responsive and user-friendly interface

## Tech Stack

### Frontend
- React (JSX)
- HTML5
- CSS3

### Backend
- Flask
- Flask-JWT-Extended (JWT Authentication)
- MySQL

## Project Structure

```
bank-management-system/
├── frontend/      # React application
├── backend/       # Flask REST API
```

## Objective

The goal of this project is to simulate the core functionality of a modern banking system while applying full-stack development best practices. It demonstrates secure user authentication with JWT, RESTful API development, database management, transaction processing, and seamless frontend-backend integration.

> **Note:** This project is developed for educational and portfolio purposes only and is not intended for production use as a real banking application.
